About IP Geolocation Tracker

A simple, developer-friendly tool to locate IP addresses and track live GPS coordinates on a map.

What it does
  • Look up geographic information (city, region, country, ISP, timezone) for any IP address.
  • Show a map marker for the IP location and a popup with details.
  • Start live GPS tracking from your device — see your moving location on the map with accuracy.
  • While live tracking runs, the information card updates coordinates and (optionally) resolves the current place.
How it works

The app uses a server-side endpoint (`/api/locate`) which calls a free IP geolocation service (ip-api.com) to retrieve IP information. The frontend displays results using Leaflet.js for interactive maps and the browser Geolocation API for live GPS tracking.

Notes & Privacy
  • The IP lookup is performed server-side to avoid exposing API keys and to respect rate limits.
  • Live GPS tracking requires browser permission; location data is handled client-side for display and not stored by the demo server.
  • Reverse geocoding (to resolve coordinates into city/region/country during live tracking) uses OpenStreetMap's Nominatim service; avoid heavy automated requests.
Run locally

From the project folder run:

python app.py

Open http://localhost:5000 in your browser.

Contact & Contribution
